Kurulum
1. Bu projeyi bilgisayarınıza indirin (Clone).
2. Klasör içindeki `AdanaCafeDB.sql` dosyasını SQL Server Management Studio'da çalıştırarak veritabanını oluşturun.
3. Visual Studio üzerinden `.sln` dosyasını açın ve projeyi çalıştırın.

4. Özellikler:
5. Kafeler türlerine göre listelenir(aile,öğrenci,çalışma ve eğlence).
6. Aradığınız kafe varsa direkt arama kısmına yazabilirsiniz.
7. Menü kısmından fiyatlara ulaşıp bütçenize en uygun yeri bulabilirsiniz.
